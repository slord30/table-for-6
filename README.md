# table-for-6
A responsive meal planning app that scales recipes for large families, provides real-time macro-nutrient data via the Edamam API, and helps parents find "kid-friendly" meals using ingredients they already have.

### Key Modules
- **Servings Scaler:** Automatically calculates ingredient amounts for a family of 6.
- **Macro Dashboard:** Displays Protein, Carbs, and Fats for every recipe.
- **Pantry Filter:** Find meals based on what's already in your fridge.
- **Favorites Manager:** Saves your top-rated recipes to `localStorage`.

### Powered By
- [Edamam Recipe Search API](https://edamam.com)
- [Edamam Nutrition Analysis API](https://edamam.com)
